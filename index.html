Server Requirements

Checking to ensure that your web host have the minimum requirements to run web services.

  • PHP >= 5.6
  • allow_url_fopen Enabled
  • short_open_tag Enabled
  • cURL support Enabled
  • PDO PHP Extension
  • JSON PHP Extension

Mail Configuration

Hosting Server:
Please open your smtp_email.php file on your web services main directory and configure with your SMTP server details
$mail->Host = 'mail.example.com'; // Specify main and backup SMTP servers
$mail->Username = 'info@example.com'; // SMTP Email (e.g. info@domain.com)
$mail->Password = 'xxxxx'; // SMTP Email password
$mail->SMTPSecure = 'tls'; // Enable TLS encryption, `ssl` also accepted
$mail->Port = 587; // TCP port to connect

If you have problem with mailing with this setup than please contact your provider with those variables and ask correct info
